# chmod package

Set file permissions.

[![Gyazo](http://i.gyazo.com/155506018c76d5a15ce056dcc28415ab.gif)](http://gyazo.com/155506018c76d5a15ce056dcc28415ab)

# Commands

* `chmod:current-file`

## Keymap

No keymap by default.

edit `~/.atom/keymap.cson`

```coffeescript
'.tree-view':
  'shift-m': 'chmod:selected-entry'
```
